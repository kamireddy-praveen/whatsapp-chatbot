# WhatsApp Chatbot (Spring Boot)

## Description
This is a simple chatbot built using Spring Boot.

## Features
- Responds to user messages
- REST API based chatbot
- Easy to extend

## Technologies Used
- Java
- Spring Boot
- REST API

## How to Run
1. Clone the repo
2. Open in VS Code / IntelliJ
3. Run the main class

## API Endpoint
POST http://localhost:8080/chat

Example request:
{
  "message": "Hello"
}
## Input Method
Input is given via REST API using tools like Postman or browser.

Example:
POST http://localhost:8080/chat
